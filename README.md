# Ashton Court Hotel - WordPress Theme

A premium, custom-designed WordPress theme for Ashton Court Hotel, Exmouth. Built with refined aesthetics, cinematic animations, and editorial-quality design.

## Features

- **Premium Design**: Light theme with warm ivory backgrounds, antique gold accents, and elegant typography
- **Smooth Scroll**: Lenis.js integration for enhanced scrolling experience
- **Cinematic Animations**: GSAP-powered scroll animations and transitions
- **Custom Cursor**: Premium cursor effects on desktop
- **Responsive Design**: Mobile-first approach with equal quality across all devices
- **Online Booking System**: Built-in room booking with availability checking
- **Custom Post Types**: Rooms, Testimonials, Events, and Team members
- **Customizer Integration**: Easy-to-use theme options via WordPress Customizer

## Installation

1. Upload the `ashton-court-theme` folder to `/wp-content/themes/`
2. Activate the theme through the 'Themes' menu in WordPress
3. Set up your menus via Appearance > Menus
4. Configure theme options via Appearance > Customize

## Page Templates

- **Front Page** (`front-page.php`): Homepage with hero, essence section, rooms preview, experiences, testimonials
- **Rooms** (`page-rooms.php`): Room listing with filters
- **Menus** (`page-menus.php`): Restaurant and bar information with menu selections
- **Weddings** (`page-weddings.php`): Wedding venue showcase and packages
- **Functions** (`page-functions.php`): Private events and corporate functions
- **About** (`page-about.php`): Hotel history and values
- **Contact** (`page-contact.php`): Contact form and location

## Custom Post Types

### Rooms
- Title, description, featured image
- Custom fields: Price, Size, Bed Type, Max Guests, View, Balcony
- Room gallery support
- Taxonomies: Room Type, Amenities

### Testimonials
- Guest name, review content, photo
- Custom fields: Location, Stay Date, Rating

### Events
- Event details and featured image
- Taxonomy: Event Type

## Theme Customization

Navigate to **Appearance > Customize** to configure:

- Hotel contact information
- Social media links
- Hero section content and media
- About section content
- Booking settings (starting price, total rooms)

## Dependencies

The theme loads the following external libraries:

- **Google Fonts**: Cormorant Garamond & Jost
- **Lenis**: Smooth scroll library
- **GSAP + ScrollTrigger**: Animation library
